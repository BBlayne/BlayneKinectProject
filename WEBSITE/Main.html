Introduction

The goal of this project is to create an application using the Windows Kinect (Version 2 for Windows 10) to create and store animations and apply them to a 3D character model in real time. The Windows Kinect is an attachment originally meant for the Xbox game console and later for the Windows PC that uses an infrared camera to read the environment and reconstruct a 3D image in order to detect hand gestures and track the movement of objects in 3 dimensions. The project will use the Windows Kinect SDK in C++ to interpret data from the Kinect and OpenGL to visualize it to the screen. This project will use the GLM library for the ease and convenience for computing matrix and vector operations. Additionally the ASSIMP (Open Asset Import Library) and its data structures will be used for the importing and management of 3D meshes and storing their recorded animations.

Features

  1. Animate a 3D Mesh using the Microsoft Kinect.

  2. Animate a 3D Mesh via manipulating the Bones of a Skeleton.

  3. Create Smooth animations through Keyframes.

  4. Change Interpolation Modes.

Goals that weren't implemented

  1. Exporting/Saving animations for future use.

Completed Goals

  1. Reading about the Theory behind the Kinect & How to use the API.

  2. Compile & Run a Sample OpenGL application that interfaces with the Kinect from the SDK.

  3. Collect & Display data from the Kinect to animate a 3D Skeleton in OpenGL.

  4. Find a mapping from imported Kinect data structures to a data structure compatible with Assimp Open Importer Library.

  5. Animated the target bones from the recorded motion from the Kinect.

  6. Animated a 3D mesh from Linear Blend Skinning using the recorded retargeted motions from the Kinect.

  7. Testing & Bug Fixing.

  8. Documentation & Demo video.

  9. Assignemnts 1 & 2.

Usage Assignment 2

This page assumes you've read "Usage (Kinect)"

Follow the same instructions in Usage (Kinect) or Usage (Assignment 1). But then select the "Interpolation" drop down menu and freely choose between Slerp, Lerp, Euler, or Matrix interpolation modes. While the Player is selected.
Remarks: Slerp/Lerp do not appear in my current implementation to show much of a different to the end user, Euler does appear to show some interesting artifacts, while Matrix for reasons explained in Interpolation.html and in Kinect.html won't show artifacts or a end user difference.
